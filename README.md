# Лендинг для ТатИТнефть
Поддерживаются светлая и темная темы.
Адаптивность под разные размеры окна/дисплея.
Модальное окно для блока "Контакты".
Tailwindcss подключен через CDN.

Для запуска проекта используется скрипт на питоне:
1. Открыть терминал в папке и написать python serverStart.py
2. Ввести адрес, который выдал скрипт
3. Если не получается подключиться - открыть порт 8000

Или

Открыть дабл кликом html файл.
